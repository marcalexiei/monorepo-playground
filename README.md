# Monorepo playground

Childish attempt to setup a decent monorepo

## Features

- ✅ npm workspace support
- ✅ TS support
  - ✅ IntelliSense support!

## Try it out

1. Clone repo
2. Run
  
   ```bash
   npm install
   ```

3. Run

   ```bash
   npm run build
   ```
